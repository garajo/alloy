Switcher



Change Theme


Theme of page changes as switcher slides. Theme is dark when switcher is off and light when on.


Disable Switcher


Both switcher and buttons are disabled to change theme.

<alloy-switcher
    [onSwitchStatus]="onSwitchStatus"
    [disabled]="switchDisabled"
    (onSwitchListener)="onSwitch($event)">
</alloy-switcher>
