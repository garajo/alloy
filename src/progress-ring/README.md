# Alloy Progress Ring

```html
<alloy-progress-ring size alloyLabel="Busy" ></alloy-progress-ring>
```

`AlloyProgressRing` is always indeterminate.

Attributes:
* icon: 32px, 5 balls.
* small: 50px, 5 balls.
* medium: 100px, 8 balls.
* large: 200px, 8 balls.
* alloyLabel: Inherits behavior from `IdentityDirective`
